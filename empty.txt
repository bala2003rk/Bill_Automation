I'm building a MERN stack Bill Reimbursement App called BillTrack. The React+Vite frontend and Node.js/Express/MongoDB backend are both complete. I need help debugging and extending.
Tech Stack:

Frontend: React + Vite @ http://localhost:5173
Backend: Node.js + Express @ http://localhost:5000/api
Database: MongoDB Atlas
Auth: JWT + bcryptjs
File uploads: Multer + Cloudinary (v1.41.3)
Email: Nodemailer (Gmail SMTP)

Roles: ADMIN, MANAGER, TEAM_LEAD, EMPLOYEE, PAYMENT_ADMIN
Bill workflow: Employee/TL submits → Manager approves/rejects → Payment Admin marks paid. Email notifications at every step.
Backend folder structure:
backend/
├── server.js
├── .env
├── config/db.js, cloudinary.js
├── middleware/auth.js          ← JWT verify + roleGuard()
├── models/User.js, Bill.js, Project.js, Department.js
├── controllers/auth, bill, user, project, department, dashboard
├── routes/auth, bills, users, projects, departments, dashboard
├── services/emailService.js   ← 4 HTML email templates
└── scripts/seed.js
Key API endpoints:

POST /api/auth/login → { token, user }
GET /api/auth/me
GET /api/bills (role-filtered)
POST /api/bills (multipart with receipt)
POST /api/bills/:id/approve|reject|pay
GET /api/dashboard/admin|manager|teamlead|employee|payment
Full CRUD: /api/users, /api/projects, /api/departments
